# Key Features

## Core POS Functionality

| Feature | Description |
| :--- | :--- |
| **Cross-Platform** | Runs as a Progressive Web App (PWA) on any modern browser, including Windows, iPad, and Android tablets. |
| **Offline-First** | All core POS operations work 100% offline. Data is stored locally and synced to the cloud when a connection is available. |
| **Multi-Tenant SaaS** | Supports multiple businesses (tenants) with strong data isolation, each with their own locations, registers, users, and roles. |

## Retail POS

| Feature | Description |
| :--- | :--- |
| **Fast Checkout** | Optimized for speed with barcode scanning, quick product search, and customizable hotkeys. |
| **Product Variants** | Manage products with multiple variants, such as size, color, and style. |
| **Discounts & Promotions** | Apply discounts to individual items or entire orders, with support for complex promotion rules. |
| **Returns & Exchanges** | Process returns and exchanges with ease, with automatic inventory updates. |
| **Layaways & On-Account Sales** | Allow customers to make partial payments or purchase items on account. |
| **Customer Accounts** | Create and manage customer profiles, with purchase history and contact information. |
| **Basic Loyalty Display** | Display customer loyalty points and tier information at the point of sale. |
| **Label & Barcode Printing** | Print product labels and barcodes directly from the POS. |

## Restaurant POS

| Feature | Description |
| :--- | :--- |
| **Floor Plan Management** | Create and manage custom floor plans with tables and seating arrangements. |
| **Table & Seat Management** | Assign orders to specific tables and seats, with support for multiple guests per table. |
| **Order & Ticket Management** | Create, modify, and track orders with a flexible ticket management system. |
| **Course Sequencing** | Organize orders by course (e.g., appetizers, main course, dessert) and send them to the kitchen in the correct order. |
| **Product Modifiers** | Customize orders with product modifiers, such as "extra cheese" or "no onions". |
| **Split & Merge Checks** | Split checks by item or by guest, and merge multiple checks into a single payment. |
| **Service Charge & Tips** | Apply automatic service charges and process tips. |
| **Takeaway & Delivery Modes** | Manage takeaway and delivery orders with dedicated workflows. |
| **KDS & Kitchen Printer Integration** | Send orders to a Kitchen Display System (KDS) or kitchen/bar printers. |

## Integrations

| Integration | Description |
| :--- | :--- |
| **Inventory** | Deep integration with Zoho Inventory for advanced inventory management, including stock movements, warehouses, and price lists. |
| **E-commerce** | Native connectors for Shopify and WooCommerce, allowing for synchronized products, inventory, orders, and customers. |
| **Loyalty** | Integration with Zoho Thrive for comprehensive loyalty programs, including points, tiers, and custom rewards. |
| **Payments** | Flexible payment processing with support for cash, bank terminals, and Yappy (Panamá). |
| **E-invoicing** | Country-specific electronic invoicing, starting with Panamá, with support for digital signatures and asynchronous processing. |
