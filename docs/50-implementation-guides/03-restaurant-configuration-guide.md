# Restaurant Configuration Guide

## 1. Overview

This guide explains how to configure a new location for a restaurant.

## 2. Steps

### Step 1: Create a Floor Plan

1.  Navigate to the "Locations" > "Floor Plans" section of the admin backoffice.
2.  Click on the "New Floor Plan" button.
3.  Enter the name of the floor plan (e.g., "Main Dining Room") and upload a background image of the floor plan.
4.  Drag and drop tables onto the floor plan to create your seating arrangement.
5.  Click "Save".

### Step 2: Create Menus

1.  Navigate to the "Products" > "Menus" section.
2.  Click on the "New Menu" button.
3.  Enter the name of the menu (e.g., "Lunch Menu") and add the items that should be included in the menu.
4.  Click "Create".

### Step 3: Create Modifiers

1.  Navigate to the "Products" > "Modifiers" section.
2.  Click on the "New Modifier" button.
3.  Enter the name of the modifier (e.g., "Extra Cheese") and the additional price.
4.  Click "Create".
