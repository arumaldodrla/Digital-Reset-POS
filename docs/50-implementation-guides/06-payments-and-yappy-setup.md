# Payments and Yappy Setup

## 1. Overview

This guide explains how to set up the payment and Yappy integration.

## 2. Steps

### Step 1: Configure Payment Methods

1.  Navigate to the "Payments" section of the admin backoffice.
2.  Enable the payment methods you want to accept (e.g., cash, credit card).

### Step 2: Configure the Yappy Connector

1.  Navigate to the "Integrations" section.
2.  Select "Yappy".
3.  Enter your Yappy merchant ID and access token.

### Step 3: Test the Integration

1.  Create a new order on the POS and select "Yappy" as the payment method.
2.  Verify that a QR code is generated.
3.  Scan the QR code with your Yappy app and make a payment.
4.  Verify that the order is marked as "completed" on the POS.
